
# BUCKLERUBY

[![CircleCI](https://circleci.com/gh/cedretaber/buckleruby.svg?style=svg)](https://circleci.com/gh/cedretaber/buckleruby)

A MinRuby interpreter written by BuckleScript.

MinRuby: A small, minimal Ruby interpreter

## Setup

```bash
$ yarn install
``` 

## Build

```bash
$ yarn build
```

## Watch

```bash
$ yarn watch
```

## Test

```bash
$ yarn test
```
